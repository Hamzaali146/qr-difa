# Contributing to QR Phish Difa

First off, thank you for considering contributing to **QR Phish Difa**! It’s people like you who make this tool better for everyone.
Your support helps improve security against phishing attacks using **Machine Learning** and **Computer Vision**.

## How Can I Contribute?

### Reporting Bugs
* Check the **Issues** tab to see if the bug has already been reported.
* If not, open a new issue. Include steps to reproduce, your Python version, and any relevant error logs from Uvicorn.

### Suggesting Enhancements
* We are always looking to improve the **ML and CV Phishing Probability** heuristics accuracy, **Integrating LLM's** and **MlOps Scalable Pipelines**
* If you have a better dataset for URL phishing, please suggest it!

### Pull Requests
1. **Fork** the repository.
2. **Create a branch** for your feature (`git checkout -b feature/AmazingFeature`).
3. **Set up the environment**:
   ```powershell
   cd backend
   python -m venv qrenv
   .\qrenv\Scripts\activate
   pip install -r requirements.txt

4. **Verify your changes:** Ensure the API starts correctly with uvicorn app.main:app.

5. Commit your changes:
    ```bash
    git commit -m 'Add some AmazingFeature'.

6. Push to the branch:
    ```bash
    git push origin feature/AmazingFeature

7. Open a Pull Request.
